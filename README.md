# Rock n Jock (Phish)

Rock n Jock is a multiplayer web app where fans predict Phish setlists and score points based on accuracy after each show.

## How It Works
- Create a game for a show date
- Share the link with friends
- Lock picks before showtime
- Setlist auto-fetches after the show
- Leaderboard updates instantly

## Scoring
- Opener — 10 pts  
- Set 1 Closer — 10 pts  
- Set 2 Opener — 10 pts  
- Set 2 Closer — 10 pts  
- Encore — 10 pts  
- Wildcards — 5 pts each  

## Data Source
Setlists and statistics powered by the Phish.net API  
https://api.phish.net/
